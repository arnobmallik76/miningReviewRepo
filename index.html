What is this?

This is the Gerrit code review datasets as part of the MSR 2016 data track paper "Mining Code Review Repositories: People, Process and Product". We extracted all the dataset from Gerrit server using our mining scripts which based on the official REST API. The data is stored as in relational database (MySQL) format.

Target Projects

We exported the database objects to self-contained files for each project. All .sql files are available here:

  • OpenStack : rar, 7z, last updated 2015/04/25
  • LibreOffice : rar, 7z, last updated 2015/05/30
  • AOSP : rar, 7z, last updated 2015/04/23
  • Qt : rar, 7z, last updated 2015/04/27
  • Eclipse : rar, 7z, last updated 2015/06/01
  • All Projects : rar, 7z.
