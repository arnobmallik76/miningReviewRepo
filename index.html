What is this?

This is the Gerrit code review datasets as part of the MSR 2016 data track paper "Mining Code Review Repositories: People, Process and Product". We extracted all the dataset from Gerrit server using our mining scripts which based on the official REST API. The data is stored as in relational database (MySQL) format.
